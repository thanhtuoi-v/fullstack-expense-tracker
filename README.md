
# Personal Expense Management Application

## Project Description

The Personal Expense Management Application helps users track their daily income and expenses, analyze spending habits, and visualize financial data through charts. It is a useful tool for managing personal finances effectively.

### Key Features:
- User registration and login
- Add, edit, and delete income/expense records
- Visual statistics by time and category
- Data visualization using charts
- Export data to Excel files
- Query data via AI chatbot 

### Technologies Used:
- **Frontend**: React, TailwindCSS, Vite, Recharts, Moment.js
- **Backend**: Node.js, Express, MongoDB, JWT, bcryptjs
- **Khác**: Multer, XLSX, dotenv

---

## Installation Guide

# Create file `.evn`
```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=8000
```

### Run backend 
```
npm install
npm run dev
```

### Run frontend
```
npm install
npm run dev
```
